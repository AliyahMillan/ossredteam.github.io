FUFF

Now that we are done with the NMAP course lets go to FUFF. FFUF, or “Fuzz Faster you Fool” is an open source web fuzzing tool, intended for discovering elements and content within web applications, or web servers. Why should we use FUFF? When visiting a website some of the web content is hidden from you. With FUFF you can uncovered anything hidden in the web applications. Good Luck and have some FUFF
